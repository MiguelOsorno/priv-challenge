
# PrivApp Backend

API to create and list new payments.

## Installation

node version 14.15.4

```bash
npm install
```

## Ejecution

```bash
npm run dev
```

## Documentation mode dev

**you need start the server**

[Api-docs](http://localhost:4000/api-docs)

## Run test

**you need start the server**

```bash
npm run test
```
